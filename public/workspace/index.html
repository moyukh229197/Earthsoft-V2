Earthsoft Usage Guide

1. Project Overview

Earthsoft provides an integrated solution for Railway Earthwork Calculations. It handles cross-sectional data, formation levels, bridges, curves, and loops to accurately compute earthwork volumes and generate BOQ estimates based on USSR rates.

2. Data Import Sequence

To build a complete model, import CSV/Excel files in this optimal order:

  1. Project Data (.ew): Load an existing project workspace.
  2. Levels Data: Contains chainage, ground levels, and proposed levels. This generates the core earthwork quantities.
  3. Bridges: Defines bridge start/end chainages. Areas intersecting bridges will be automatically handled as deductions in the calculation engine.
  4. Curves: Defines railway curve radii. This applies extra formation width on curves.
  5. Loops & Platforms: Applies additional track centers and platform widths for station yards.

3. Calculation Methodology

Volumes are calculated using the Trapezoidal Rule applied sequentially between adjacent cross-sections. When structures (Bridges, Stations, Curves) overlap, the calculation engine dynamically inserts sub-chainages to segment the earthwork calculation perfectly along structure boundaries.

Volume = (Area1 + Area2) / 2 * Distance

4. Verifying & Exporting

Use the Verify button in the top toolbar to validate the calculation engine outputs. Once validated, you can view the 'Calculation Table', visualizations, and 'Estimates' tabs. Use Export Report to generate a comprehensive PDF calculation sheet.
